# callAll

HOC that call all given functions.

import:
```typescript copy
import { callAll } from '@fullstacksjs/toolbox'
```

usage:
```typescript copy
callAll(...functions)
```

example:
```typescript copy
  callAll(foo, bar, baz)(...args)                         // void (foo(...args), bar(...args), baz(...args));
  callAll(bar,'string', 5, null, undefined, baz)(...args) // void (bar(...args), baz(...args));
```
