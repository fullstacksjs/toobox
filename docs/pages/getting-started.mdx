# Getting Started

## Installation

### Node

npm

```shell copy
npm add --save-dev @fullstacksjs/toolbox
```

pnpm

```shell copy
pnpm add --save-dev @fullstacksjs/toolbox
```

yarn

```shell copy
yarn add --dev @fullstacksjs/toolbox
```


### Deno

```typescript copy
import * as toolbox from 'https://deno.land/x/toolbox/mod.ts'
```

### Browser ESM

ES Module

```html copy
<script type="module" src='https://deno.land/x/toolbox/mod.ts'></script>
```

Script

```html copy
<script src='https://www.unpkg.com/@fullstacksjs/toolbox/iife/index.js'></script>
```
