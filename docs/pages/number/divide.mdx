# divide

Divides two numbers returns fallback if result is not a finite number

### Import

```typescript copy
import { divide } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function divide(dividend: number, divisor: number, fallback: number = NaN): number {}
```

### Examples

```typescript copy
divide(4, 2)     // 2
divide(4, 0, 42) // 42
divide(4, 0)     // NaN
```
