# toDecimal

Parses the string to decimal and returns fallback when the value is not finite

### Import

```typescript copy
function toDecimal(s: string, fallback: number = NaN): number {}
```

### Signature

```typescript copy
function toDecimal(
  min: number = 0,
  max: number = Number.MAX_SAFE_INTEGER,
): number {}
```

### Examples

```typescript copy
toDecimal('S'); // NaN
toDecimal('100'); // 100
toDecimal('1.42'); // 1
toDecimal('S', 10); // 10
toDecimal('S', undefined); // NaN
```
