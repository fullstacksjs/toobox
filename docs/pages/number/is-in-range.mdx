# isInRage

Checks if num is between min and max (including borders).

### Import

```typescript copy
import { isInRage } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function isInRange(num: number, min: number, max: number): boolean {}
```

### Examples

```typescript copy
isInRange(51, 0, 50)   // false
isInRange(-1, 0, 100)  // true
isInRange(100, 0, 100) // true
isInRange(0, 0, 100)   // true
isInRange(50, 0, 100)  // true
```
