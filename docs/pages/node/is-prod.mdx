# isProd

Check whether NodeJS Env is production.

### Import

```typescript copy
import { isProd } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function isProd(): boolean {}
```

### Examples

```typescript copy
/* NODE_ENV=development node */
isProd(); // true

/* NODE_ENV=production node */
isProd(); // false

/* NODE_ENV="" node */
isProd(); // false
```
