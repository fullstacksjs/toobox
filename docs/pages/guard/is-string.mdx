# isString

check given value is string or not.

### Import
```typescript copy
import { isString } from '@fullstacksjs/toolbox';
```

### Signature
```typescript copy
isString(value): boolean
```

### Examples
```typescript copy
isString()       // false
isString(1)      // false
isString(['1'])  // false
isString('1')    // true
```
