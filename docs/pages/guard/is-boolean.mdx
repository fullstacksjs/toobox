# isBoolean

check given value is boolean or not.

### Import
```typescript copy
import { isBoolean } from '@fullstacksjs/toolbox';
```

### Signature
```typescript copy
isBoolean(value): boolean
```

### Examples
```typescript copy
isBoolean(true)   // true
isBoolean(false)  // true
isBoolean()       // false
isBoolean(1)      // false
isBoolean(['1'])  // false
isBoolean('1')    // false
```
