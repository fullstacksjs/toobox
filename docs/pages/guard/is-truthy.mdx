# isTruthy

check given value is truthy.

### Import
```typescript copy
import { isTruthy } from '@fullstacksjs/toolbox';
```

### Signature
```typescript copy
isTruthy(value): boolean
```

### Examples
```typescript copy
isTruthy(undefined) // false
isTruthy(null)      // false
isTruthy(0)         // false
isTruthy(false);    // false
isTruthy({});       // true
isTruthy([]);       // true
isTruthy(NaN);      // false
```
