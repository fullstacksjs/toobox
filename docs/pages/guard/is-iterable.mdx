# isIterable

check given value is iterable or not.

### Import
```typescript copy
import { isIterable } from '@fullstacksjs/toolbox';
```

### Signature
```typescript copy
isIterable(value): boolean
```

### Examples
```typescript copy
isIterable([])                            // true
isIterable(new Map())                     // true
isIterable({ [Symbol.iterator]() { } })   // true
isIterable('some')                        // true
isIterable({})                            // false
isIterable()                              // false
```
