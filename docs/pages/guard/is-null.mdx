# isNull

check given value is either null or undefined.

### Import
```typescript copy
import { isNull } from '@fullstacksjs/toolbox';
```

### Signature
```typescript copy
isNull(value): boolean
```

### Examples
```typescript copy
isNull(undefined) // true
isNull(null)      // true
isNull(0)         // false
isNull(false);    // false
isNull({});       // false
isNull([]);       // false
```
