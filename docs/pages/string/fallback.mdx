# fallback

Returns the provided fallback value if the given value is null, undefined, or an empty string; otherwise, it returns the original value.

### Import

```typescript copy
import { fallback } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function fallback<T extends string = string>(value: Nullable<T>, fallbackValue: T): NonNullable<T> {}
```

### Examples

```typescript copy
fallback('', 'd')        // 'd'
fallback(' ', 'd')       // ' '
fallback('x', 'd')       // 'x'
fallback(null, 'd')      // 'd'
fallback(undefined, 'd') // 'd'
```
