# isLastIndex

is an index is the last index in an array

import:
```javascript copy
import { isLastIndex } from '@fullstacksjs/toolbox'
```

usage:
```javascript copy
isLastIndex(array, index)
```

example:
```javascript copy
isLastIndex([], 0)            // false
isLastIndex([1], 0)           // true
isLastIndex([1, 2], 1)        // true
isLastIndex([], -1)           // false
isLastIndex([[], [], []], 0)  // false
isLastIndex([[], [], []], 2)  // true
isLastIndex([undefined], 0)   // true
isLastIndex([null], 0)        // true
isLastIndex([,], 0)           // true
isLastIndex([, null], 1)      // true
isLastIndex([, undefined], 1) // true
```
