# compact

Filters nullish items from an array

### Import

```typescript copy
import { compact } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function compact<T extends readonly unknown[]>(xs: T): FilterNullish<T> {}
```

### Examples

```typescript copy
compact([]); // []
compact([1]); // [1]
compact([0, '', null, false, NaN, undefined]); // [0, '', false, NaN, {}]
```
