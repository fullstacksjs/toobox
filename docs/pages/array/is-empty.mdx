# isEmpty

Return whether an array is empty or not

### Import

```typescript copy
import { isEmpty } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function isEmpty(x: unknown[]): boolean {}
```

### Examples

```typescript copy
isEmpty([])            // true
isEmpty([1])           // false
isEmpty([undefined])   // false
isEmpty([null])        // false
isEmpty([,])           // false
isEmpty([, null])      // false
isEmpty([, undefined]) // false
```
