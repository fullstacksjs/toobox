# toArray

Convert a value to an array.

import:
```typescript copy
import { toArray } from '@fullstacksjs/toolbox'
```

usage:
```typescript copy
toArray(value)
```

example:
```typescript copy
toArray(null)        // []
toArray(undefined)   // []
toArray('string')    // ['string']
toArray(['array'])   // ['array']
toArray(iteratable)  // [...iteratable]
toArray(others)      // [others]
```
