# getRandom

get a random element from an array.

### Import
```typescript copy
import { getRandom } from '@fullstacksjs/toolbox';
```

### Signature
```typescript copy
getRandom(value)
```

### Examples
```typescript copy
getRandom([1, 2, 3])    // 1 or 2 or 3
getRandom([])           // undefined
getRandom('list')       // Error: not an array
```
