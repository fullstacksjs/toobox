# shuffle

Gets an array and return a shuffled version of it.

import:
```typescript copy
import { shuffle } from '@fullstacksjs/toolbox'
```

usage:
```typescript copy
shuffle(array)
```

example:
```typescript copy
copyArray([]);                // []
copyArray([1]);               // [1]
copyArray([1, 2, 3 ,4 , 5]);  // [2, 1, 3, 4, 5]
```
