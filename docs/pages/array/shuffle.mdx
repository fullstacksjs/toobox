# shuffle

Return a shuffled version of an array

### Import

```typescript copy
import { shuffle } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function shuffle<T extends any[]>(arr: T): T {}
```

### Examples

```typescript copy
shuffle([]); // []
shuffle([1]); // [1]
shuffle([1, 2, 3]); // Who knows?
```
